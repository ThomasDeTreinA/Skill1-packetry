## 🔌 Thuis Spelen & Installatie

Wil je zelf ervaren hoe het is om stapsgewijs een netwerk op te bouwen? Je kunt Packetry heel eenvoudig lokaal op je eigen computer draaien via Python en Pygame.

### Systeemeisen
* Geschikt voor Windows, macOS of Linux.
* **Python 3.10** of nieuwer geïnstalleerd.

### Stap-voor-stap handleiding
1. **Download het project:** Klik bovenaan op deze GitHub repository op de groene knop `Code` en kies `Download ZIP`.
2. **Pak de ZIP-omgeving uit** op een handige locatie op je computer.
3. **Installeer de libraries:** Open je terminal of opdrachtprompt (CMD) in de uitgepakte map en voer uit:
   ```bash
   pip install -r requirements.txt
