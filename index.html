🔌 Thuis Spelen & Installatie

Wil je zelf ervaren hoe het is om stapsgewijs een netwerk op te bouwen? Je kunt Packetry heel eenvoudig lokaal op je eigen computer draaien via Python en Pygame.

Systeemeisen

Windows, macOS of Linux met Python 3.10 of nieuwer geïnstalleerd.

Stap-voor-stap handleiding

  1. Download het project: Klik bovenaan op onze GitHub repository op de groene knop Code en kies Download ZIP.
  2. Pak de ZIP-omgeving uit op een handige locatie op je computer.
  3. Installeer de libraries: Open je terminal/opdrachtprompt (CMD) in de uitgepakte map en voer uit:
pip install -r requirements.txt

4. Start de simulator: Lanceer de applicatie direct met het commando:

python main.py

📚 Level Overzicht

De simulator bevat 4 opeenvolgende levels die naadloos aansluiten op de officiële onderwijsdoelen van de 3e graad.

Level Thema Wat je leert & ontdekt
Level 1 🔌 De Basis (LAN) Gebruik van PC's, Hubs en Switches. Werken met Straight-Through kabels en kabelafstanden (Cat 5 vs Cat 5e).
Level 2 🛠️ Routing & ISP Handmatige IP-configuratie, Subnet Masks en het instellen van ISP-gegevens op de router.
Level 3 🏢 DHCP & Crossover Blueprint-weergave, netwerkredundantie met routers (via Crossover) en DHCP-configuratie.
Level 4 🌍 Wi-Fi & Wereldkaart Draadloze Wi-Fi netwerken opzetten op een Laptop en gebouwen overbruggen via WAN Fiber.

💬 Veelgestelde Vragen (FAQ)

Voor wie is deze simulator bedoeld?
Packetry is specifiek ontworpen als een intuïtieve visuele brug voor studenten uit de 3e graad secundair onderwijs (Informaticawetenschappen / STEM). Het helpt leerlingen om de eindtermen rondom datacommunicatie sneller te verwerken via actieve gamificatie.
Waarom verschilt Packetry van Cisco Packet Tracer?
Cisco Packet Tracer is gebouwd voor professionals en heeft een steile leercurve. Packetry daarentegen biedt direct 'pick-up-and-play' comfort. Je ziet datapakketjes fysiek stromen; gaat er iets fout, dan stopt de stroom visueel. Dit is perfect voor beginners.
In welke stack is dit ontwikkeld?
De volledige simulator is geprogrammeerd in Python met gebruik van de Pygame library. De modulaire opbouw zorgt ervoor dat elk netwerkcomponent (Switch, Router, PC) functioneert als een configureerbare class.
